# Math Magicians <a name="about-project"></a>

Math Magicians a Single Page App (SPA) that allows users to use a calculator, and read a random math-related quote. This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app). This is the first project in Module-3 of the Full Stack Developer program at Microverse.

## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- <a href="https://nodejs.org/en/download">Node.js</a>
- <a href="https://docs.npmjs.com/downloading-and-installing-node-js-and-npm">npm</a>

### Setup

Clone this repository to your desired folder:

```sh
  cd desired-folder
  git clone https://github.com/henokkhm/microverse-math-magicians.git
```

### Install

Install this project with:

```sh
  cd microverse-math-magicians
  npm install
```

### Usage

- To start the app in the development mode on your local machine, simply run:

`npm start`

Then open [http://localhost:3000](http://localhost:3000) to view it in your browser. The page will reload when you make changes. You may also see any lint errors in the console.

- To run the linters, run: 

`npm run lint`

### Run tests


- To run the tests, run the following script from the root of the project: 
  
`npm test`

This will launch the test runner in the interactive watch mode.\
See the section about [running tests](https://facebook.github.io/create-react-app/docs/running-tests) for more information.


### Deployment

To build static assets for deployment, run the following script from the root of the project:

`npm run build`

The static assets will be saved in the `build` folder. The build is minified and the filenames include the hashes. Your app is ready to be deployed!
